# UI 风格规范（清新治愈）

## 设计关键词
清新、轻松、低负担、可呼吸感。

## 色彩
- 背景：#F7FAF8
- 主色：#63C7A3
- 辅色：#A7E3CF
- 文字主色：#1F2937
- 次级文字：#6B7280
- 警示：#F59E0B

## 组件
- 卡片圆角：16
- 按钮圆角：14
- 卡片阴影：轻（y:4 blur:12 opacity:0.08）
- 间距：8/12/16/24 体系

## 文案风格
- 简短、温柔、明确
- 示例：
  - "出发前看一眼，今天更安心。"
  - "拿了吗？钥匙、钱包、耳机。"

## 动效
- 轻微淡入、缩放过渡（0.2~0.3s）
- 不使用复杂弹跳
