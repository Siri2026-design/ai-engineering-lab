# 拿了吗（TakeIt）1.0 PRD

## 1. 目标
帮助出行用户在离开关键地点时，提醒随身物品，减少遗忘与丢失。

## 2. 范围（1.0）
- iOS 原生 App（SwiftUI）
- 本地数据存储，不上传云端
- 地点离开提醒（GeoFence + 本地通知）
- 物品清单管理
- 清新治愈风 UI

## 3. 核心功能
1) 物品清单
- 默认项：手机、钥匙、钱包、耳机、工牌
- 支持增删改

2) 地点管理
- 支持 1~3 个地点（家/公司/自定义）
- 每个地点配置提醒半径（默认 5m）

3) 离开提醒
- 监控离开地点事件
- 触发通知："拿了吗？请确认：钥匙、钱包..."
- 可点击“已带齐”关闭当次提醒

4) 可用性保护策略
- 用户默认半径 5m
- 若定位精度不佳（horizontalAccuracy > 20m），临时使用 30m 作为触发兜底，防止漏提醒

## 4. 非功能要求
- 响应快、流程短（3 步内完成设置）
- 通知文案简短、低打扰
- 电量友好：依赖系统地理围栏回调，不做高频后台轮询

## 5. 验收标准
- 能添加物品、添加地点
- 真机离开地点后能收到通知
- 通知内容包含当前启用物品
- UI 为清新治愈风（浅色背景+圆角卡片+统一留白）
- 全数据仅本地存储

## 6. 已知限制
- iOS 地理围栏并非厘米级，不保证 5m 精准触发
- 室内定位波动会影响触发时机（已用 30m 兜底）
