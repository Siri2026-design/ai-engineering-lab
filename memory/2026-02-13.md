## Feishu 回消息间歇性失效（复发）排障记录

- 现象：飞书偶发“收到消息但不回 / 回应延迟”。
- 已确认根因之一（当天已修复）：`feishu` 插件重复加载导致冲突告警（duplicate plugin id）。
  - 内置路径：`/opt/homebrew/lib/node_modules/openclaw/extensions/feishu`
  - 用户扩展路径（旧）：`~/.openclaw/extensions/feishu`
- 处理动作：
  1. 备份并清理 `~/.openclaw/openclaw.json` 中的 `plugins.installs.feishu`
  2. 将用户扩展目录中的 feishu 移出 `~/.openclaw/extensions/`（放到 `_disabled_plugins`）
  3. 重启 gateway 并复查
- 修复后状态：
  - `openclaw plugins list` 显示仅单一 `feishu`（loaded）
  - `openclaw status --deep` 中 Feishu 为 `ON / OK`
  - 权限检查 `feishu_app_scopes`：33 granted, 0 pending
- 复发判断：后续短时“又没回复”更像链路抖动/事件投递波动，不是重复插件问题。
